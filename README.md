# 원티드 프리온보딩 3주차 - 한국임상정보 검색 영역 클론 코딩

## 📚 과제

### 한국임상정보 웹사이트의 검색영역 클론 코딩

#### 과제1. 검색창 구현

- 키보드만으로 추천 검색어들로 이동 가능하도록 구현

#### 과제2. 검색어 추천 기능 구현

- 사용자의 입력값에 따른 추천 검색어 제공
- API 호출 횟수를 줄이는 전략 수립 및 실행

#### 과제3. 캐싱 기능 구현

- API 호출별로 로컬 캐싱 구현
- 캐싱 기능을 제공하는 라이브러리 사용 금지(React-Query 등)
